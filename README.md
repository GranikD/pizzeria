# pizzeria
Это код, написанный на Питоне, представляющий собой консольную пиццерию, в которой пользователь вводит свое имя и возраст(которые в последствии сохраняются в базу данных с другими пользователями), в соответствии с которым он получает меню, в котором может совершить заказ.
В коде присутствуют комментарии, но если есть проблемы с пониманием я прикрепил презентацию по проекту, где есть блок-схема с возможно более понятным объяснением.
